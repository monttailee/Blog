#### 假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

#### 举栗
```
输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
```

#### 解题
```
var climbStairs = function (n) {
    let dp = new Array(n);
    dp[1] = 1;
    dp[2] = 2;
    for (let i = 3; i <= n; i++) { // 到达第n阶楼梯有从n-1阶走一步和从第n-2阶走两步两种情况
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
};
```
